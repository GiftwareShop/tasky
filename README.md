## Getting Started

### Prerequisites

- Node.js (v18+)  
- npm or yarn  

### Installation

```bash
1. Clone the repository:
git clone https://github.com/GiftwareShop/tasky.git
cd tasky

2. Install dependencies:
npm install
# or
yarn install

3. Start the development server:
npm run dev
# or
yarn dev

4. Open your browser and go to http://localhost:5173
```
