## Getting Started

### Prerequisites

- Node.js (v18+)  
- npm or yarn  

### Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/tasky.git
cd tasky

2. Install dependencies:
npm install
# or
yarn install

3. Start the development server:
npm run dev
# or
yarn dev

4. Open your browser and go to http://localhost:5173
